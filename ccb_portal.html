Local Setup Required

Directive 0: Environmental Preparation

Before beginning integration tests, ensure your local terminal is synchronized with the CCB Core Services.

"We cannot trust the cloud. Your local sandbox must be isolated and dependencies strictly controlled."
  1. Clone the Archive:
    git clone https://github.com/cr0wnf0x/coruscant-wero-sandbox-demo.git cd coruscant-wero-sandbox-demo
  2. Initialize Core Logic:
    python3 -m venv .venv source .venv/bin/activate pip install flask lxml requests mcp
  3. Ignite Background Services:

    Run these in two separate background terminal tabs:

    # Tab 1: Clearing Gateway python3 mock_gateway.py
    # Tab 2: Fraud Oracle (SSE) python3 mcp_fraud_server.py
  4. Register Oracle:
    gemini mcp add --transport sse fraud-oracle http://localhost:8000/sse
